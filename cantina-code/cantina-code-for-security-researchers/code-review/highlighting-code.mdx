# ðŸ”¸ Highlighting code

Upon **highlighting a portion of code,** it is possible to do the following options after clicking the 3 horizontal dots toggle:

* **Copy Line**: Copies the contents of the highlighted area.
* **Copy Permalink**: Copies the URL pointing to the highlighted area.
* **Comment Line**: Opens a comment under the highlighted area.
* **Link to Existing Finding**: Links the highlighted area to an already open finding.
* **Reference in New Finding**: Starts new finding submission process.
