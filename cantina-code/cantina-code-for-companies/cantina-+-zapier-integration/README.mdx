### Overview

Connect Cantina with your favorite tools using [Zapier](https://zapier.com/). With this integration, you can automate workflows by triggering actions in other apps when key events happen in Cantinaâ€”no code required.

Whether you're triaging security findings, alerting your Ops or Security team, or syncing updates to Slack or Discord, Zapier helps you stay informed and responsive.

***

### Key Features

* **Trigger Zaps from Cantina** when:
  * A new finding is submitted
  * A label like "Escalated" or "Client opinion needed" is added
* **Send alerts** to Slack, Discord, PagerDuty, Opsgenie, email, and more
* **Notifications are instant**
* **Self-serve API key management** for secure and reliable integrations

***
