{% hint style="info" %}
If a core team member has registered an account for you,  follow the steps to [claim-an-account.md](../claim-an-account.md "mention").
{% endhint %}

Follow the steps below to **create a new account**.

1. Navigate to [https://cantina.xyz/signup](https://cantina.xyz/signup) and introduce your email and new password of choice.

<figure><img src="../../.gitbook/assets/image (26).png" alt="Cantina security platform login screen." width="294"><figcaption><p>Signup modal</p></figcaption></figure>

2. Verify your email and select "**I am a security researcher**".

<figure><img src="../../.gitbook/assets/image (27).png" alt="Security platform role selection screen." width="367"><figcaption><p>Account selection</p></figcaption></figure>

3. Chose the public username of your choice and introduce an **invitation code**. Signups are currently invite-only. If you don't have an invite code yet, follow and tweet at [@cantinaxyz](https://twitter.com/cantinaxyz) on Twitter or ask an existing user for one.

{% hint style="info" %}
Your username will be displayed to all users _e.g.,_ [_cantina.xyz/u/noah_](https://cantina.xyz/u/noah)_._ Profile visibility can be changed in profile settings.
{% endhint %}



<figure><img src="../../.gitbook/assets/image (20).png" alt="Web3 security platform researcher registration." width="292"><figcaption><p>Security Researcher signup</p></figcaption></figure>

{% hint style="success" %}
Now you are all set to use the Cantina application!
{% endhint %}
