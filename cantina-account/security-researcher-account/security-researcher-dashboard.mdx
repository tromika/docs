# ðŸ”¹ Security Researcher Dashboard

The Security Researcher Dashboard shows you important information at a glance.

* **Profile information** where you can edit your public profile, manage your invitation codes and payments.
* **Edit your availability** [calendar.md](calendar.md "mention").
* **Notifications** triggered by events such as Cantina Repository creation or [comments-and-pings.md](../../cantina-code/cantina-code-for-security-researchers/comments-and-pings.md "mention"), also received via email.
* **Access to** active and past Security Reviews and Competitions.
